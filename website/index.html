About This Investigation

RBI has created a dense network of subsidiaries, departments, and controlled entities — IDRBT, IFTAS, ReBIT, RBIH, NPCI, and its own IT and Fintech departments — that operate critical national payments and banking infrastructure. These entities share personnel, board seats, and institutional culture with no external accountability.

The .bank.in domain portal — the exclusive gateway for all 1,500+ Indian banks to register under their mandated .bank.in domain — was awarded to a private company without competitive tender. It went live without a security audit. When tested, it revealed 33+ unauthenticated API endpoints, 5,576 exposed records, and 1,072 orphan Super Admin accounts.

This is not a security failure. It is a governance failure that is structural, not accidental.
